收費會員網站，包含會員付費、購買課程、註冊等功能，並使用排程完成月費自動扣款。

主要技術為 **Nuxt** x **Laravel** x **Tailwindcss** 。

在本站開發扮演吃重角色，負責網站前後端及大部分介面設計。參與細節包含 Api 撰寫與串接、金流串接、流程與 UI 設計。
